# XRetrieval: Advanced Information Retrieval Search Engine

## 📌 Project Overview

**XRetrieval** is a robust backend-focused search engine developed as part of the **Information Systems & Retrieval** curriculum at the Faculty of Computers and Artificial Intelligence, Fayoum University. The system is designed to efficiently index, process, and retrieve relevant documents using advanced NLP pipelines and diverse search algorithms.

## 🚀 Key Features

* **Comprehensive NLP Pipeline:** Implements sophisticated Tokenization and Stemming to normalize text data.
* **Advanced Indexing:** Utilizes both **Inverted Index** and **Positional Index** structures for optimized retrieval performance.
* **Diverse Search Capabilities:**
* **Boolean Search:** For precise logical query matching.
* **Phrase Search:** To find exact sequences of terms using positional data.
* **Phonetic Search (Soundex):** To retrieve results based on phonetic similarity, enhancing user experience for misspelled queries.


* **RESTful API Architecture:** Scalable backend core logic built with .NET Web APIs.

## 🛠️ Tech Stack

* **Language:** C#
* **Framework:** .NET Core / Web API
* **Core Concepts:** Information Retrieval (IR), Natural Language Processing (NLP), Data Structures & Algorithms.
